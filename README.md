🛒 Amazon Sales Dashboard
📌 Short Description / Purpose

The Amazon Sales Dashboard is a powerful and interactive Power BI report designed to analyze Amazon sales performance across multiple categories, states, and customer segments. It provides a complete overview of revenue, purchase cost, profit margins, and product trends to help businesses make smarter, data-driven decisions.

This dashboard is ideal for e-commerce analysts, business owners, sales managers, and data enthusiasts who want to track profitability, identify top-performing products, and understand regional sales patterns.

🚀 Tech Stack

The dashboard was built using the following tools and technologies:

📊 Power BI Desktop – Main platform used to create dynamic and interactive dashboards.
📂 Power Query – Used for cleaning, transforming, and preparing raw sales data.
🧠 DAX (Data Analysis Expressions) – Used for KPIs, custom measures, and profit calculations.
📝 Data Modeling – Relationships created between sales, category, state, and customer data tables.
📁 File Format – .pbix for development and .png for dashboard preview.
📂 Data Source

The dataset contains Amazon sales transaction records including:

Product Name
Category
Purchase Cost
Sale Price
Profit
Gender-wise Sales
State-wise Orders
Date / Day-wise Transactions

This structured dataset helps uncover business performance trends across products and regions.

✨ Features / Highlights
📉 Business Problem

E-commerce businesses generate huge volumes of sales data daily, but raw spreadsheets make it difficult to identify:

Which products generate maximum profit?
Which states drive the most revenue?
Which category performs best?
Are male or female customers contributing more to sales?
How do daily sales fluctuate over time?

Without visualization, these insights are hard to detect quickly.

🎯 Goal of the Dashboard

To build an interactive dashboard that:

Tracks total sales, purchase cost, and profit instantly
Identifies best-selling categories and products
Monitors regional sales distribution across India
Compares customer sales by gender
Analyzes day-wise revenue trends for better forecasting
